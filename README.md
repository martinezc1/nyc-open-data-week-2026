# NYC Open Data Week 2026

## Teaching with NYC Open Data: Publishing Student Civic Research Through Reproducible Workflows

This repository contains materials for our NYC Open Data Week 2026 session featuring the **NYC Open Data Student Gallery**, a collection of original civic research projects conducted by graduate students at Brooklyn College.

Students used real NYC Open Data datasets, the [nycOpenData](https://martinezc1.github.io/nycOpenData/) R package, and fully reproducible workflows in R and Quarto to produce public-facing research chapters now published as an open educational resource.
